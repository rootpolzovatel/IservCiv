## Программа определения возраста пользователя:

```
currentYear = 2025
currentMonth = 10
currentDay = 25

birthYear = int(input("Введите год вашего рождения: "))
birthMonth = int(input("Введите месяц вашего рождения: ")) 
birthDay = int(input("Введите день вашего рождения: "))

age = currentYear - birthYear 

if(currentMonth < birthMonth):
    age -= 1
elif(currentMonth == birthMonth):
    if(currentDay < birthDay):
        age -= 1

print(f"Ваш возраст: {age} лет")
```

## Практика с циклом **WHILE**

```
userInput = 1
proizvedenie = 1

while userInput != 0:
    proizvedenie *= userInput
    userInput = int(input("Введите ваше число: "))

print(f"Произведение ваших чисел равно: {proizvedenie}")
```
